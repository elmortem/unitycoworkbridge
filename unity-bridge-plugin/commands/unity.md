# /unity

Выполнить скрипт в открытом Unity Editor через Cowork Bridge.

## Использование

Эта команда — точка входа для выполнения любых операций в Unity Editor. Пользователь или Claude вызывает её, когда нужно создать, модифицировать или проанализировать что-то в открытом Unity проекте.

## Выполнение

Передай управление скиллу `unity-bridge` (SKILL.md) для выполнения задачи.

## Аргументы

Команда принимает описание задачи на естественном языке. Например:

```
/unity добавь компонент Rigidbody на все объекты с тегом Enemy
```
